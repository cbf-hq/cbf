# CBF Chromium Patch Series

This directory stores the exported CBF patch files applied to `chromium/src`.
These patch files contain diffs against Chromium source files.
See `LICENSE.chromium` in the repository root for the Chromium source license
terms that apply to Chromium-derived portions of these patches.

For patch queue order, patch-splitting principles, and workflow policy, see:

- [Chromium Fork Workflow](../../../docs/book/src/developer-guide/chromium-fork-workflow.md)
