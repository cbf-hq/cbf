# CBF Book

CBF (Chromium Browser Framework) is a Rust-oriented browser backend framework built on Chromium.

This book is the canonical reader-facing entry point for:

- application developers integrating CBF
- contributors working on the framework itself
- maintainers documenting architecture and boundary rules

The current structure intentionally separates:

- quick onboarding for users
- architecture and conceptual overview
- implementation constraints for contributors

As the book grows, existing repository docs will be folded into these chapters in a more guided order.
