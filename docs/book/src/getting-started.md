# Getting Started

Use this book as the primary documentation entry point.

Recommended reading order:

1. Read [User Setup](./user-setup.md) if you want to use prebuilt CBF artifacts.
2. Read [Architecture](./architecture.md) to understand the framework's layering.
3. Continue to contributor chapters only if you will modify the framework or Chromium bridge.

The existing `README.md` remains the repository landing page.
This book is intended to become the canonical long-form guide.
