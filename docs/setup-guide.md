# CBF Setup Guide (Overview)

Choose the setup path based on your role:

- `docs/user-setup-guide.md`
    - For application developers using prebuilt CBF and `cbf_bridge` artifacts.
    - No local Chromium build required.
- `docs/developer-setup-guide.md`
    - For contributors building or modifying CBF/Chromium bridge locally.
    - Includes `depot_tools` and Chromium build steps.

Related:

- `docs/architecture.md` for design intent and layering.
- `docs/chromium-fork-guide.md` for fork-specific layout and patch policy.
- `docs/implementation-guide.md` for bridge/FFI implementation rules.
